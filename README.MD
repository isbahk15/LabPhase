AgroLoop - Agricultural Waste Marketplace
AgroLoop is a full-stack MERN application designed to connect farmers and merchants. It allows users to list agricultural waste, tools, and seeds for sale, promoting a circular economy in the agricultural sector.

=> Features
User Authentication: Secure registration and login using JWT (JSON Web Tokens) and bcrypt password hashing.

Marketplace: A real-time searchable grid of agricultural listings.

Dynamic Categories: Filterable items including Organic Waste, Seeds, Tools, and Recyclable Plastics.

Merchant Profiles: Every listing is linked to a verified merchant account.

Contact Integration: Direct WhatsApp integration for buyers to reach sellers instantly.




ech Stack
Frontend
React.js: Functional components and Hooks.

Axios: For API communication.

React Router: For navigation between Marketplace, Dashboard, and Product Details.

Backend
Node.js & Express: API routing and server logic.

MongoDB & Mongoose: NoSQL database for flexible data modeling.

JWT: Secure, stateless authentication middleware.
Dotenv: Environment variable management.
